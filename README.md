# Job Tracker (MERN Stack)

A full-stack Job Tracker application built using MongoDB, Express, React, and Node.js.

## Features
- JWT Authentication (Login/Register)
- Add/Edit/Delete Jobs
- Job Stages: Applied, Interview, Offer, Rejected
- Search and Filters
- Drag-and-Drop Kanban Board
- Interview Reminders
- Email Notifications
- Resume Upload
- Analytics Dashboard
- Export PDF / CSV

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Auth: JWT
- Storage: Cloudinary
- Notifications: Nodemailer

## Folder Structure
- `client/` → React frontend
- `server/` → Express backend
